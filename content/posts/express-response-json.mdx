---
title: Express send JSON
date: 2020-05-12
excerpt: You can write simply response JSON with Express JS
---

After you have setup connection with route in Express app, the callback function will call
Request object instance and Reponse object intance as below:

```js
app.get('/', (req, res) => res.send('Hello Express'))
```

And you can directly send `JSON` to the client by using simple code as follow
```js
app.get('/', (req, res) => res.json({ name: "Hello", message: "Express" }))
```
