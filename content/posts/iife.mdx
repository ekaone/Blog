---
title: IIFE
date: 2020-05-01
excerpt: IIFE (Immediately Invoked Function Expression)
---

An IIFE (Immediately Invoked Function Expression) is a JavaScript function that runs as soon as it is defined.

```js 
(function() {
  // ...your codes
})();
```
...or use arrow function `() =>`
```js
(() => {
  // ...your codes
})();
```
